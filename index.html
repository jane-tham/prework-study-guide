HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Best practice is to use semantic elements whenever possible.
  • Metadata is information that helps the browser render the page correctly.
  • Use the headings tags in order from h1 to h6, making sure h1 only appears once per page.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet and external CSS style sheet.
  • Remember to link the external CSS style sheet file to the html file to see the changes.
  • You can style HTML elements using elements as selectors or by using the class attribute as selectors.
  • When styling a class selector in CSS, it is denoted by using a period first before the class name.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches to the main branch
  • git pull origin main: pulls the most updated version of the project from the GitHub repo to your local machine
